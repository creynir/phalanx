# Terminology Refactor Mapping

This document specifies the mandatory terminology mappings for the Phalanx project.
All codebase files, documentation, specs, and tests must align with these terms.

- **Advisor** ➡️ **Team Lead**
- **Replanner / Strategist** ➡️ **Engineering Manager**
- **Sequencer** ➡️ **Orchestrator**
- **Blueprint** ➡️ **Workflow**
- **Skill** ➡️ **(Deleted/Removed)** (Use Workflow directly)
- **Step** ➡️ The node on the Workflow graph (an orchestration wrapper around a Block)
- **Block** ➡️ The execution engine inside a Step
